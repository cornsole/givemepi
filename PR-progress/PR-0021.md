   - block의 `[start, end)` 범위와 tree level

   - signed magnitude와 minimal big-endian magnitude 적용
   - zero·양수·음수·대형 정수 round-trip
   - leading zero와 잘못된 sign 조합 거부

3. Versioned P/Q/T block codec 구현

   - magic·version·header size·flags 정의
   - fixed-width integer의 explicit big-endian encoding
   - identity·range·level·P/Q/T metadata와 payload encoding
   - checked size arithmetic와 deterministic byte output

4. CRC32C checksum 구현

   - checksum algorithm ID와 length 기록
   - canonical header와 전체 payload 보호
   - fixed fixture와 single-bit mutation 검증

5. In-memory codec 검증

   - P/Q/T와 metadata round-trip
   - encode → decode → encode byte equality
   - 잘못된 길이·범위·비정규 encoding 거부

6. Atomic file commit 구현

   - 동일 directory의 unique temp file 사용
   - short write 처리와 file sync

   - 실패 시 기존 final file 보존 및 fault injection 검증

7. CheckpointStore 구현

   - codec과 filesystem storage 책임 분리

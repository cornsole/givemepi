  1. TaskSharedState 도입

      - exception_ptr
      - mutex와 condition variable
      - Task와 TaskHandle이 동일 상태 공유

  2. TaskHandle 재정리
      - shared_ptr<Task> 제거
      - 공유 상태만 참조
      - move-only 유지
      - valid(), wait()
      - 준비·완료·실패 상태 조회
      - 저장된 예외 확인·재전달 기능

  3. Task::execute() 연결
      - 실행 전후 상태 변경

      - 예외 저장 후 Failed 처리

  4. 제출 API 통일
      - ThreadPool::submit(Task) → TaskHandle

  1. Chudnovsky Binary Splitting 계약 확정

  - Binary Splitting 범위는 [start, end)로 통일
  - start >= end인 빈 범위와 역전 범위는 예외로 거부
  - 범위 검증은 unsigned 범위 연산과 재귀 진입 전에 수행
  - k = 0 leaf는 P=1, Q=1, T=13591409로 정의
  - k > 0 leaf는 Chudnovsky P/Q/T 공식 적용
  - 홀수 k의 T는 음수, 짝수 k의 T는 양수
  - leaf 계수는 GMPInteger로 변환한 뒤 계산하여 중간 정수 overflow 방지
  - 기존 merge 방정식 유지
  - 인접한 [a, m), [m, b) 노드만 [a, b)로 병합
  - split()은 splitSequential()에 위임
  - splitParallel()은 동일한 범위 계약과 수학적 결과를 유지
  - scheduler 실행·cutoff 정책·parallel merge는 PR-0019로 연기

  2. GMP 정수 연산 보강

  - Chudnovsky leaf 계산에 필요한 부호 반전 연산 추가
  - GMPInteger 외부에서 raw GMP 연산을 직접 사용하지 않도록 캡슐화
  - k 관련 계수를 GMPInteger로 변환한 뒤 계산
  - 기본 정수에서의 중간 곱셈 overflow 방지
  - 양수·0·음수 부호 반전 테스트 추가

  3. 입력 범위 검증 구현
